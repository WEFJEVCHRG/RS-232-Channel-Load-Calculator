# RS-232 Channel Load Calculator

Web tool for calculating RS-232 serial communication parameters.

## Features
- Calculate transmission time per packet
- Analyze channel load percentage
- Dynamic table for multiple packet types
- Real-time calculations

## Live Demo
https://WEFJEVCHRG.github.io/RS-232-Channel-Load-Calculator/

## Usage
1. Enter packet parameters
2. Add multiple packet types
3. Click "Calculate"
4. View results instantly

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
